# txt2png

This command convert txt file to png.

## Usage

```
txt2png <filename> <width> <height>
```

You can get <filename>.png in current directory.

## Example

```
$ git clone https://github.com/onokatio/txt2png
$ cd txt2png
$ chmod +x ./txt2png
$ ./txt2png ~/COJT/hoge/imagedata.txt 640 480
```
